# Инструкция для работы с Markdown

## Выделение текста

Чтоб выделить текст курсиво необходимо обрамить его звездочками (*). Например, *Вот так*.

Чтобы выделаить текст полужирным, необходимо обрамить его двойными звездочками (**).
